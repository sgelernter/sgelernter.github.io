Portfolio

A few recent projects:

"Watch That Mix!"

"Watch that mix!" is a fun, educational resource built with JavaScript and utilizing DOM manipulation to create a dynamic, engaging experience where users can learn a bit about what goes into professionally mixing songs.

Bandplace: "A Place for Bands"

A full-stack clone of artist storefront/streaming platform Bandcamp. Built using Ruby on Rails and PostgreSQL for the back-end/database and React/Redux for the front-end.

SynthGarden

A MERN-stack web application to make music with an in-browser synth instrument. Users can explore synth settings, create patches, and record short performance samples to download for later.
